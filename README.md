# Orlando Handyman Services Landing Page

This is a professional landing page for a handyman service business in Orlando, Florida.

## Features

- Responsive navigation with mobile menu
- Hero section with call-to-action buttons
- Services section showcasing different handyman services
- About section with company information and values
- Testimonials from satisfied customers
- Contact form with service area information
- Professional footer with social links

## Technologies Used

- Next.js (Pages Router)
- TypeScript
- Tailwind CSS
- React

## Components

All components are located in the `src/components` directory:

- `Navigation.tsx` - Responsive navigation header
- `Hero.tsx` - Hero section with headline and call-to-action
- `Services.tsx` - Grid of services offered
- `About.tsx` - Company information and values
- `Testimonials.tsx` - Customer testimonials
- `Contact.tsx` - Contact form and information
- `Footer.tsx` - Footer with links and copyright

## Development

To run the development server:

```bash
npm run dev
```

To build for production:

```bash
npm run build
```

To start the production server:

```bash
npm run start
```

## Deployment

This site is configured for deployment on Vercel.
